Perguntas Frequentes sobre Códigos de Bancos

O que é o código do banco (COMPE)?

O COMPE, ou Código de Compensação, é um número de três dígitos que identifica cada banco no Brasil. Ele é essencial para realizar transferências como TED e DOC, e para o processamento de cheques.

Qual a diferença entre COMPE e ISPB?

COMPE é o código de 3 dígitos para compensação. ISPB (Identificador do Sistema de Pagamentos Brasileiro) é um código de 8 dígitos, mais detalhado, também usado para identificar instituições financeiras no sistema de pagamentos. Para a maioria das transações de varejo (TED, DOC), o COMPE é o mais utilizado.

Como encontrar o código do meu banco?

Você pode usar nossa ferramenta de busca nesta página digitando o nome do banco. O código também pode ser encontrado no seu talão de cheques, no internet banking, no aplicativo do seu banco, ou em extratos bancários.

Esta lista de códigos de bancos está atualizada?

Nós nos esforçamos para manter a lista de códigos dos bancos brasileiros o mais atualizada possível, baseando-nos em informações públicas e de fontes confiáveis como o Banco Central do Brasil. No entanto, devido a possíveis fusões ou alterações, recomendamos sempre confirmar com seu banco ou no site oficial do BACEN para transações críticas.
